# Goal

## Project
KnowledgeBot — a python-based intelligent Q&A platform with RAG capabilities.

## Description
KnowledgeBot is a platform for building enterprise-grade AI agents with Retrieval-Augmented Generation (RAG) capabilities. It allows users to:
- Create and manage knowledge bases by uploading documents
- Automatically split documents into chunks and create vector embeddings for semantic search
- Configure AI applications/agents with custom prompts and settings
- Chat with agents that retrieve relevant knowledge to answer questions
- Support multiple LLM providers (OpenAI, etc.)

Key features:
- User authentication and management
- Knowledge base management with document upload and processing
- Text chunking and embedding generation
- Vector similarity search for relevant context retrieval
- Application/agent configuration with RAG settings
- Chat interface with streaming responses
- REST API for all operations

## Scope
- ~25-30 production source files to implement
- ~10-15 test files to write
- Reproduce core RAG platform functionality with Django backend
